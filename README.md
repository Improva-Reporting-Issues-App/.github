# Improva

**A better space, together.**

Improva is a geolocated issue reporting platform for large and shared spaces.

It helps organizations collect, manage, assign, and resolve maintenance issues across environments such as:

- campuses
- universities
- municipalities
- marinas
- sports facilities
- resorts
- large private venues

## What Improva does

- Map-based issue reporting
- Precise geolocation
- Photo upload
- Multilingual reporting flows
- Centralized issue management
- Assignment and status tracking
- Support for public and private spaces
